▷ 선택자
	선택자란? 스타일을 지정할 요소를 특정하는 문법이다.	
	여러가지 방식이 존재한다. 

▶ 기본 선택자
	전체 선택자: *
	태그 선택자: 태그 이름 ex. div, p ...
	아이디 선택자: #속성값(id 이름) -> id 속성값을 이용하여 지정. 하나의 html 문서 안에서 고유한 값이어야 한다.
	클래스 선택자: .속성값 -> id와 다르게 중복해서 사용할 수 있다. 한 번에 여러 곳에 지정한 후 한번에 스타일 지정 가능
	속성 선택자: 선택자[속성]: 태그의 해당 속성이 있는 요소 선택
		 선택자[속성=값]: 태그의 해당 속성과 값이 일치하는 요소 선택
