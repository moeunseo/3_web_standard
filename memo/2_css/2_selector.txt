▷ 선택자
	선택자란? 스타일을 지정할 요소를 특정하는 문법이다.	
	여러가지 방식이 존재한다. 

▶ 기본 선택자
	전체 선택자: *
	태그 선택자: 태그 이름 ex. div, p ...
	아이디 선택자: #속성값(id 이름) -> id 속성값을 이용하여 지정. 하나의 html 문서 안에서 '고유한 값'이어야 한다.
	클래스 선택자: .속성값 -> id와 다르게 중복해서 사용할 수 있다. 한 번에 여러 곳에 지정한 후 한번에 스타일 지정 가능
	속성 선택자: 선택자[속성]: 태그의 해당 속성이 있는 요소 선택
		 선택자[속성=값]: 태그의 해당 속성과 값이 일치하는 요소 선택

▶ 조합 선택자 > 기본 선택자를 조합해서 사용!
	그룹 선택자: 선택자, 선택자, 선택자, ...{} -> 선택자에는 기본 선택자들 중(전체 선택자 제외) 아무거나 와서 그룹화시킴!
	자식 선택자: 부모 선택자 > 자식 선택자 -> 하나의 태그 안에 수많은 태그들을 사용할 수 있다. 즉, 수많은 태그들 중 또 다르게 
				       스타일을 지정하고 싶을 때 사용
	하위 선택자(손주라고 생각): 선택자 선택자 선택자 -> 자식에만 한정이 아닌 손주, 중손주까지 전부 적용(잘 사용하진 않음)
	형제 선택자1: 선택자 + 선택자 -> 첫번째 선택자 다음으로 오는 형제 요소에만 적용
	형제 선택자2: 선택자 ~ 선택자 -> 첫번째 선택자와 형제인 요소들에게 모두 적용

▶ 가상 요소 선택자
	HTML 문서에 실제 작성된 요소는 아니지만, 존재하는 것처럼 취급해 선택하는 지정방법을 말한다.
	쉽게 말해 선택한 요소의 일부에만 스타일을 적용하는 건데 새로운 요소를 추가한 것처럼 동작한다.
	실무에서 많이 사용되는 두가지만 알아보자!

	선택자::before{}, 선택자::after{}
	
	각각 해당 선택자의 앞 공간, 뒷 공간 선택할 수 있다.
	content속성을 활용하여 새로운 텍스츠 추가 및 해당 텍스트에 스타일 지정 가능

▶ 가상 클래스 선택자
	요소의 상태를 이용하여 선택자로 지정하는 방식

	선택자:가상 클래스 선택자{}
	
	1) 동적: 어떤 행동에 따라 변하는 상태에 따라 선택자를 지정할 수 있다. -> 많이 사용하는 친구!
		-:active : 요소를 마우스를 클릭하고 있는 동안 선택자로 지정 
		-:hover : 마우스가 쓰윽 위로 올릴 때 선택자로 지정

	2) a 태그: 링크의 방문여부에 따라 선택자를 지정할 수 있다.
		- :link: 방문하지 않은 링크일 때 선택자로 지정
		- :visited: 방문한 적 있는 링크일 때 선택자로 지정

	3) input 태그: 상호 작용 요소 상태에 따라 선택자를 지정할 수 있다.
		- :focus: 요소가 선택될 때 선택자로 지정
		- :checked: 체크박스가 체크되어 있을 경우 선택자로 지정
		- :disabled: disabled 속성이 부여되어있을 때 선택자로 지정
		- :enabled: disabled가 아닐 때 선택자로 지정

	4) 구조적 가상 클래스

▶ CSS 적용 우선순위
	일단 태그들마다 고유하게 부여되어있는 스타일이 존재한다.
	스타일을 어디든 어떻게 선택해서 지정하든 고유하게 부여되어있는 스타일은 사라진다.

	우선순위가 같다면 '마지막'으로 읽힌 스타일이 적용된다. (단계적 적용)
	
	- 선택자별 우선 순위 정리 (오른쪽은 점수)
	인라인 스타일	1000
	아이디		100
	가상 클래스 	10
	클래스 		10
	가상 요소		1
	태그		1
	전체		0

	굳이 이 점수를 디테일학 외우고 계산할 필요없이, 자식 태그로 내려가는 것 하위 요소로 찾아가는 것 등등으로
	선택자를 선택한다면 더 우선순위가 높다~ 정도의 느낌만 가져도 충분!
