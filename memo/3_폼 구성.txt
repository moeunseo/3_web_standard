▷ 폼 구성하기
	폼은 HTML에서 클라이언트 데이터를 수집하기 위해 사용한다.
	클라이언트에게 정보를 입력받고 서버로 전송하기 위한 양식이라고 생각!
	서버로 데이터를 전송하는 주요 수단이며, 여러 속성들이 존재
	다양한 HTML 요소를 포함할 수 있다.

▶ form 태그 > 백엔드로 데이터를 보낼 때 무조건 사용!
	전체적인 폼 양식을 의미한다. 폼을 구성하는 모든 요소는 form 태그 안에 작성한다.
	form 태그 안에서 입력하는 모든 걸 서버 쪽으로 전송한다.

	<form action="서버 주소(url)" method="전송 방식">
		폼을 구성하는 요소 // 입력받아야 할 데이터
	</form>

	action
		클라이언트가 입력한 폼 요소 안에 데이터를 전송할 url 주소를 작성한다.
		서버를 배우기 전까진 #을 채워놓자!

	method
		입력받은 데이터를 서버로 전송할 때 전송 방식을 작성한다.

▶ input 태그
	가장 보편적이고 기본적인 데이터를 입력받는 태그
	
	<input type="" name="" value="">

	type: 필수 속성, 어떤 유형으로 데이터를 입력 받을지를 정할 수 있다.
	name: 해당 input 태그의 이름을 작성한다. 나중에 서버에서 해당 태그의 데이터를 받을 때, name 속성 이름으로 데이터를 관리
	value: 초기 값 작성

▶ label 태그
	상호 작용 요소에 이름을 붙일 때 사용한다.
	label에 작성해준 이름만 클릭하더라도 요소 선택이 가능하다.
	시각장애인이 웹을 이용할 때 사용하는 보조 도구인 스크린 리더기가 label 태그로 연결된 상호 작용 요소를 쉽게 식별할 수 있어서
	웹 접근성 향상을 위해서라도 필수로 사용하길 권장한다.

▶ fieldset, legend 태그
	fieldset 태그로 form 태그 내에 여러 상호 작용 요소들을 그룹으로 묶을 수 있다.
	legend 태그를 이용해서 각 그룹의 이름을 붙일 수 있다.

	<form>
		<fieldset>
			<legend>이름 설정</legend>
				각종 상호 작용 요소들 추가
		</fieldset>
	</form>

▶ textarea 태그
	input 태그는 기본적으로 한줄로 입력하는 태그이다.
	여러 줄의 입력 요소를 생성할 때 사용한다.

	<textarea>초기값</textarea>
	
▶ select 태그
	셀렉 박스를 만들어주는 태그이다.
	option과 optgroup 요소를 포함할 수 있다.

	<select>
		<optgroup label="이름">
		<option value="전송할 값">화면에 뿌려줄 텍스트</option>
		</optgroup>
	</select>

	option 태그: 셀렉박스에 목록을 추가할 때 사용하는 태그
	optgroup 태그: 목록들을 그룹화할 수 있다. label 속성을 상용해서 그룹의 이름도 지정해줄 수 있다.

	select 태그의 속성들
		size: 화면에 보이는 목록의 갯수
		multiple: 여러 개의 선택가능 하도록 해주는 속성(crtl누르면 됨)
		selected: 화면을 처음 접속할 때 기본값을 설정(이미 선택되어 있음) > 많이 사용하는 친구

▶ button 태그
	input 타입 button과 별도로 button 태그가 존재한다.

	<button type="타입">버튼내용</button>
	-> 이미지를 버튼으로 사용하고 싶을 때 많이 사용

	submit: 서버에 내용을 전송하기 위한 버튼
	reset: 입력된 내용 초기화 버튼
	-> form태그 안에서 사용하는 속성들

	button: 일반 버튼
	-> 보통 이전/다음 페이지로 갈 때 사용

▷ 폼 구성하기 관련 추가 속성들

▶ disable, readonly 속성
	disable: 비활성화, 모든 상호 작용 요소에 사용 가능.
	readonly: 읽기 전용, input(text, password,...), textarea에서만 사용 가능

	두 개의 차이
	disable을 준 요소는 서버로 데이터가 넘어가지 않고,
	readonly 인 요소의 데이터는 넘어간다.

	<input ... disabled>, <input ... readonly) 등등으로 사용

▶ maxlength 속성
	글자 수 제한
	텍스트 관련된 요소들에 걸어줄 수 있는 속성이다.

	<textarea maxlength="숫자(글 제한 수)"> </textarea>

▶ ★☆placeholder 속성 -> 가장 많이 사용할 것으로 추정!
	상호작용 요소에 입력할 값에 대한 가이드를 클라이언트에게 제공할 수 있다.

	<input type="email" placeholder="ex.mes98@naver.com">

▶ checked 속성
	체크박스 타입의 상호작용 요소에 이미 체크가 된 상태로 표시할 수 있다.

	<input type="checkbox" checked>
