▷ 내장 객체
▶ String: 문자열을 다룸
	속성
		length: 길이 반환
	메소드	
		includes('문자열') -> 포함한다면 true, 아니라면 false
		slice(시작, 종료직전) -> 지정된 범위 추출
		split('구분자') -> 문자열을 구분자를 기준으로 분할 후 배열 반환
		trim() -> 문자열의 양 끝 공백 제거 후 반환

▶ String: 배열을 다룸
	속성
		length: 길이 반환
	메소드
		push(요소) -> 요소 추가
		pop() -> 마지막 요소 삭제
		forEach(함수) -> 배열의 각 요소마다 함수 호출
		filter(함수) -> 배열의 각 요소마다 콜백함수 호출 후 true를 반환하는 요소를 가지고 새 배열 생성
		-> 갯수 변화O, 값 변화X
		map(함수) -> 배열의 각 요소마다 콜백함수 호출 후 결과를 요소로 하는 새 배열 생성 
		-> 갯수 변화X, 값 변화O
