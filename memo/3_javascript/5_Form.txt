▷ 폼 조작하기

▶ form 태그 선택하기
	1) document의 forms 속성을 이용 > html문서 안에 모든 form 속성을 가지고옴
		- document.forms[0]

	☆2) form태그의 name 속성을 이용하여 선택
		- document.loginForm

▶ form태그 내 요소 선택하기
	1) elements로 요소 배열 가지고오기
		- document.loginForm.elements[0]
		- document.loginForm.elements['idInput'] // name이 idInput인 요소

	☆2) 요소의 name 속성을 이용하여 선택 
		- document.loginForm.idInput // name이 idInput인 요소

▶ 입력 요소 내의 값 가져오기
	value 속성
		- document.loginForm.pwInput.value.length

	이 방법을 이용하여 유효성 검사 진행!

▶ 기타
	checked 속성: checkbox, radio의 옵션 선택 제어
	selected 속성: select 태그의 옵션 선택 제어
